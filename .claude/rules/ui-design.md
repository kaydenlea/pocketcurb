# UI Design Rule

Gama design must feel calm, premium, minimal, useful, and consistency-first.

## Required Reads

- `docs/agent-workflows/ui-design-standard.md`
- `docs/product/shared/product-thesis.md`

## Guardrails

- avoid one-off styling and random visual experiments
- optimize for clarity, context, and suggestions rather than guilt or raw data overload
- keep mobile product UX and web marketing design concerns separate
- use Stitch, Gemini image generation, 21st.dev, and similar tools for ideation only, not as the source of truth
- do not send sensitive product or user data to external design tools
- record external-reference provenance in the active spec or review notes when a shipped UI borrows a material pattern
