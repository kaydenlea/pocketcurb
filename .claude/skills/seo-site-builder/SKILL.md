# SEO Site Builder

1. Read `docs/agent-workflows/seo-standard.md`, `docs/product/web/seo-content-strategy.md`, and `docs/product/shared/product-thesis.md`.
2. Build from real product truth rather than generic budgeting claims.
3. Avoid unsupported feature promises and disclosure-risk copy.
4. Keep SEO analytics aligned with privacy and disclosure rules.
5. Preserve the separation between web acquisition strategy and mobile product decisions.
6. Update content strategy docs when a stable SEO lesson emerges.
