# Feature Inventory

## MVP Features

1. Daily Spending Meter.
2. Add or Simulate Transaction.
3. Daily Budget Setting.
4. Today's Transactions List.
5. Event or Trip Budget with automatic event grouping.
6. Auto-assign transactions to an event, allow edits, and flag ambiguous cases.
7. Split option for shared transactions.
8. Forward-looking week or month view for insights, budgets, payments, and cash-flow visibility.
9. Safe-to-Spend number.
10. Privacy toggle on shared categories or contexts.
11. Rollover budgeting.

## Later Features

1. Multiple currencies.
2. Voice or whisper-style transaction capture.
3. Budget modes, including FIRE-oriented modes.
4. Gap-fill behavior that connects FIRE planning to daily budgeting.
5. Real-time long-term impact and FIRE-date movement.
6. Exportable reports to Excel or Sheets.
7. Shared use for couples, families, or multiple people.
8. Investments and assets.
9. Running balances to prevent overdrafts and improve planning.
10. Long-term goal framing for daily purchases.
11. Freelancer or entrepreneur mode with personal vs business separation.
12. Tax-bracket projections and quarterly estimate support.
13. Privacy-first local-storage mode with strong manual entry and text or SMS transaction capture options.
14. Spreadsheet import.
15. HYSA rate tracking and suggested cash moves.
16. Transaction export and customizable reports.

## Feature Ordering Logic

The first release sequence should prioritize features that reduce daily friction, improve trust, and fix common spreadsheet failure modes before expanding into advanced planning.
